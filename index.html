Cidadestados.org

API pública para persistir dados das cidades e estados brasileiros.

O que é?

Para que você não ter que reinventar a roda toda vez que for criar um aplicativo que exija a persistência entre as cidades e os estados brasileiros, o Cidadestados.org disponibiliza esta API pública para que você tenha, através do formato JSON, toda a informação que precisar.

Como fazer?

Faça uma requisição do tipo GET para um dos endereços a seguir e você terá uma resposta do tipo JSON prontinha para ser iterada:

  • Lista de todos os estados:

    http://cidadestados.org/estados/

  • Informações de um estado (por UF), incluindo suas cidades:

    http://cidadestados.org/estados/sp/
